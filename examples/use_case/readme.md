# 示例

我们在 `examples` 目录中放置了一些示例。所有示例都使用与 [Manus](https://manus.im/?utm_source=ai-bot.cn) 相同的提示词。

我们使用的模型是 `claude3.5`。
## 机票查询
使用携程查询最近一个月从上海到香港的低价机票

## 日本旅行计划
**提示词**：
```
我需要一个从上海出发的7天日本行程（1月15-23日），预算为15000-30000元，适合我、妻子、7岁儿子。我们喜欢历史遗迹、隐藏的宝藏和日本文化（剑道、茶道、禅修）。我们想看看奈良的鹿，并步行探索城市。我计划在这次旅行中求婚，需要一个特殊的地点推荐。请提供详细的行程安排和一个简单的HTML旅行手册，包含地图、景点描述、基本日语短语和旅行提示，以便我们在整个旅程中参考。
```
